Создание новой категории

Новая категории

Отображаемое имя категории может быть записано в виде обычного текста для каждой вновь созданной категории или с помощью указанных ниже Параметров.

Параметры, которые могут быть использованы для построения названия:

  • ● {ageGroup}: `взрослые`, `до 18`, `до 16`, `до 12`
  • ● {seniors}: `да`, если `возраст от` больше или равен 35, иначе `нет`
  • ● {adults}: `да`, если `возраст от` больше или равен 18, иначе `нет`
  • ● {juniors}: `да`, если `возраст до` равен 18, иначе `нет`
  • ● {team}: `да`, если категория предназначена для команды, иначе `нет`
  • ● {type}: `ката`, `кумитэ`
  • ● {sex}: `Женский`, `Мужской`, `Смешанный`
  • ● {age}: Генерируемое именование в зависимости от `пола` и установки `возрастной категории`
  • ● {categoryAgeName}: Значение настройки возрастной категории `Название`
  • ● {ageFrom}: Значение установки возрастной категории `Возраст от`
  • ● {ageTo}: Значение установки возрастной категории `Возраст до`
  • ● {weight}: Генерируемый предварительный просмотр веса в зависимости от настройки `Весовой категории` (для KATA вместо веса будет показан `KATA`)
  • ● {weightFrom}: Значение настройки возрастной категории для `Вес от`
  • ● {weightTo}: Значение установки весовой категории `Вес до`
  • ● {grades}: Предварительный просмотр оценок категории
